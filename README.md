# Ex-4 Scenario-Based Report Development Using Prompt Engineering Techniques


## Aim

To develop a scenario-based report using advanced prompt engineering techniques and evaluate outputs from multiple AI tools.

---

## Experiment

Generate a report using the following prompting techniques:

1. Comparative Analysis Prompt  
2. Experiential Perspective Prompt  
3. Everyday Functioning Prompt  
4. Universal Prompt  
5. Prompt Structure Refinement  
6. Prompt Size Limitations  

Use at least two AI tools (ChatGPT, Gemini, etc.) and compare outputs.

---

## Algorithm

1. Select a scenario (e.g., AI in education or healthcare).  
2. Create prompts for each technique.  
3. Execute prompts in two AI platforms.  
4. Record outputs from both tools.  
5. Compare outputs based on quality and clarity.  
6. Analyze differences between tools.  
7. Document findings in report format.  
8. Convert to PDF and submit.  

---

## Prompting Techniques with Examples

### 1. Comparative Analysis Prompt

**Prompt:**  
Compare AI in education and healthcare.

**Output:**  
AI provides comparison in structured format.

---

### 2. Experiential Perspective Prompt

**Prompt:**  
Explain AI usage from a teacher’s perspective.

**Output:**  
AI gives explanation based on role.

---

### 3. Everyday Functioning Prompt

**Prompt:**  
Explain how AI helps in daily life.

**Output:**  
Simple real-life examples provided.

---

### 4. Universal Prompt

**Prompt:**  
Explain AI for beginners.

**Output:**  
General explanation understandable to all.

---

### 5. Prompt Structure Refinement

**Prompt (Refined):**  
Explain AI with examples, advantages, and limitations.

**Output:**  
Detailed structured answer.

---

### 6. Prompt Size Limitation

**Prompt:**  
Explain AI in 3 lines.

**Output:**  
Concise answer.

---

## Output

A comparative report showing how different prompt techniques and AI tools perform.

---

## Result

Thus, scenario-based prompt engineering techniques were applied and evaluated successfully across multiple AI platforms.
